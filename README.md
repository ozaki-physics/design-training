## 使い方 Usage
メモ書きや、様々なサイトのコピペ、自分で試してみたものを保存する<br>
本当なら feature/内容ブランチ を作って develop ブランチにマージして開発しないといけないが<br>
面倒だから develop ブランチに直接コミットして、気が向いたら main ブランチとマージする<br>
個人利用しかしませんが、著作権等に配慮したいのでコピペされて困るなどあれば教えてください

## 参考文献 References
